Docker Image

  • docker image build --tag=IMAGE_NAME .: build an image, IMAGE_NAME should be lowercase
  • docker image ls: show all images
  • docker image rm IMAGE_NAME: remove an Image
  • docker container ls: show all activ containers
  • docker container ls -a: show all containers
  • docker container stop CONTAINER_NAME: stop a container
  • docker container rm CONTAINER_NAME: remove a container
  • docker run --name CONTAINER_NAME -p PORT_OUT:PORT_IN IMAGE_NAME: create a container for an image and lauch it to PORT_OUT

Docker Compose

  • docker compose up: Started all services
  • docker compose up -d: Started all services in background
  • docker compose build: build the images in Service
  • docker compose build --no-cache: build the images in Service and ignore cache
  • docker compose down: Stop all Services
